# Theory Validation

**Status**: theory and protocol audit
**Question**: which parts of the mathematical frame are defensible given the
current evidence, estimators, and runtime protocol?

## What It Contains

This folder keeps theorem/protocol analysis, estimator checks, verifier-noise
experiments, context-pressure analysis, and retained theory artifacts. It is not
an empirical success claim.

## Main Result

The original claim was too broad. The current defensible position is narrower:
a single-axis theorem with explicit assumptions and a Jensen remainder. The
empirical validation remains incomplete.
