# Figures

Public figures for the agent memory ablation study. They are generated by
`scripts/plot_agent_memory_ablation.py` from the preserved probe summary JSON.

These figures use the valid-analysis set only for outcome claims. Probes `P05`-
`P08` are excluded because memory was configured but silently broken.

| file | message |
| --- | --- |
| `figure-01-probe-outcomes.png` / `.pdf` | best validation BPB reached by valid primary probes |
| `figure-02-memory-stabilization.png` / `.pdf` | shared memory stabilizes exploratory agents compared with no-memory controls |
| `figure-03-validity-filter.png` / `.pdf` | which probes are valid, excluded, context-only, or not run |
