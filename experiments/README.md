# Experiments

This directory contains curated experiment evidence for AgentOps Lab.

`passes/` keeps result bundles that support the evaluation docs under
[`docs/evals/`](../docs/evals/) and the research protocol under
[`docs/research/`](../docs/research/). In the showcase export, prompt-history,
task-runner internals, integration logs, and pass-level process scripts are
excluded so the public tree stays focused on results and reproducibility.
